# A.I.D. - Agência de Investigação de Dados 🕵️‍♂️📊

Produto Educacional desenvolvido para o **Projeto Integrador IV - Licenciatura em Matemática (UNIVESP 2026)**.

## 🎯 Objetivo Pedagógico
Um jogo web gamificado focado no ensino de **Probabilidade e Estatística** para o 9º ano do Ensino Fundamental, alinhado à Base Nacional Comum Curricular (BNCC).
## 📚 Habilidades BNCC Trabalhadas
* **Caso 01 (Cofre Digital):** (EF09MA20) Análise de eventos dependentes e independentes em experimentos aleatórios.
* **Caso 02 (Fraude Midiática):** (EF09MA21) Análise crítica de gráficos e identificação de manipulações visuais.
* **Caso 03 (Painel de Controle):** (EF09MA22) Escolha e construção do gráfico mais adequado para cada tipo de dado.
* **Caso 04 (Pesquisa de Campo):** (EF09MA23) Planejamento de pesquisa amostral e cálculo de medidas de tendência central (Média, Moda, Mediana e Amplitude).

## 🛠️ Tecnologias Utilizadas
* HTML5, CSS3, JavaScript (Vanilla).
* Biblioteca Chart.js (para renderização de gráficos).
* LocalStorage (para salvamento de progresso local).

## 🚀 Como Executar
1.  Acesse o link oficial: [https://github.com/profsergioericmatematica/Projeto_Integrador_IV_UNIVESP_Jogo_AID_Agencia_de_Investigacao_de_Dados/](https://profsergioericmatematica.github.io/Projeto_Integrador_IV_UNIVESP_Jogo_AID_Agencia_de_Investigacao_de_Dados/)
2.  Ou baixe este repositório e abra o arquivo `index.html` no seu navegador.
